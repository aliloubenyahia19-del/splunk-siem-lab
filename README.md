# Splunk SIEM Lab

## Overview
This project demonstrates basic use of Splunk for monitoring and analysing logs. The goal was to understand how Security Information and Event Management (SIEM) tools help detect suspicious activity.

## Objectives
- Learn how Splunk collects and displays log data
- Analyse logs to identify unusual behaviour
- Understand basic security monitoring concepts

## Tools Used
- Splunk
- Sample log data
- Basic queries (search commands)

## Activities Performed
- Explored Splunk interface and dashboards
- Searched and filtered log data
- Identified patterns in system activity
- Investigated potential suspicious events

## Example Use Cases
- Monitoring login attempts
- Detecting unusual activity patterns
- Reviewing system logs for anomalies

## What I Learned
- How SIEM tools are used in security operations
- Basic log analysis techniques
- Importance of monitoring for early threat detection

## Future Improvements
- Learn advanced queries
- Simulate real attack scenarios
- Integrate more data sources
